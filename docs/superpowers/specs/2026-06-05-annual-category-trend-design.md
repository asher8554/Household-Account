# 연간 카테고리별 소비 변화 설계

## 목표

`연간 소비 추세` 화면에서 월별 총지출뿐 아니라 카테고리별 소비가 1년 동안 어떻게 변했는지 볼 수 있게 한다.

## 승인된 접근

사용자가 승인한 방식은 `누적 막대 + 요약 목록`이다.

월별 누적 막대 차트는 각 월의 지출이 어떤 카테고리로 구성됐는지 보여준다. 카테고리별 요약 목록은 연간 총액, 비율, 최고 지출월, 최근 3개월 증감을 보여준다.

## 범위

- 기존 `연간 소비 추세` 화면에 새 섹션을 추가한다.
- 지출 거래만 카테고리 변화 집계에 사용한다.
- 수입과 순액 표시는 기존 월별 상세 영역에 둔다.
- 상위 8개 카테고리를 개별 표시하고 나머지는 `기타 묶음`으로 합친다.
- 저장 동작이나 데이터 모델 변경은 추가하지 않는다.

## 검증

- 카테고리별 집계 함수 단위 테스트를 먼저 작성한다.
- `npm run build`로 타입 체크와 프로덕션 빌드를 확인한다.
- Playwright로 로컬 렌더링, 탭 전환, 차트 표시, 콘솔 오류 없음을 확인한다.
